### Frontend (chocu-fe)

-- NODE version 18

#### Dependencies:

1. @fortawesome/free-solid-svg-icons: Cung cấp icon FontAwesome dạng solid.
2. @fortawesome/react-fontawesome: Hiển thị các icon FontAwesome trong React.
3. @reduxjs/toolkit: Công cụ quản lý state với Redux, tối ưu reducer và slice.
4. @tippyjs/react: Tạo tooltip đẹp mắt và dễ dàng tích hợp vào React.
5. antd: Thư viện UI mạnh mẽ cho React với các component như Button, Table, Form, Modal.
6. axios: HTTP client để gọi API, hỗ trợ interceptors và xử lý lỗi.
7. date-fns: Thư viện xử lý ngày tháng hiện đại và nhẹ.
8. immer: Quản lý immutable state trong Redux.
9. joi: Validate dữ liệu (form hoặc input).
10. lodash: Thư viện utility với các hàm xử lý object, array, string.
11. path: Thư viện làm việc với đường dẫn file trong Node.js.
12. react: Thư viện chính xây dựng giao diện người dùng với component-based architecture.
13. react-dom: Quản lý render các component React vào DOM.
14. react-helmet: Quản lý metadata của trang (SEO, tiêu đề).
15. react-icons: Bộ icon phổ biến như FontAwesome, Material Icons cho React.
16. react-redux: Kết nối Redux với React (useSelector, useDispatch).
17. react-router-dom: Quản lý điều hướng (routing) trong React.
18. react-slick: Thư viện carousel (slider) trong React.
19. react-toastify: Thư viện hiển thị toast notifications.
20. slick-carousel: CSS và JavaScript cho slider của react-slick.
21. uuid: Thư viện tạo UUID cho các object hoặc key.
22. @eslint/js: Cấu hình ESLint cho JavaScript.
23. @types/react: Type definitions cho React trong TypeScript.
24. @types/react-dom: Type definitions cho react-dom.
25. @vitejs/plugin-react: Plugin cho Vite giúp tích hợp React.
26. autoprefixer: Công cụ tự động thêm tiền tố CSS.
27. eslint: Công cụ kiểm tra mã nguồn JavaScript/TypeScript.
28. eslint-plugin-react: Quy tắc ESLint cho mã nguồn React.
29. eslint-plugin-react-hooks: Quy tắc ESLint cho React hooks.
30. eslint-plugin-react-refresh: Hỗ trợ React Fast Refresh trong quá trình phát triển.
31. globals: Cung cấp các global variables cho JavaScript.
32. postcss: Công cụ xử lý CSS với khả năng mở rộng mạnh mẽ.
33. sass: Preprocessor CSS giúp viết mã dễ dàng và mạnh mẽ.
34. sass-embedded: Hỗ trợ biên dịch Sass trong môi trường IDE và build systems.
35. sass-loader: Loader cho Webpack để xử lý file Sass.
36. tailwindcss: Framework CSS utility-first giúp xây dựng UI nhanh chóng.
37. vite: Build tool và bundler nhanh cho dự án JavaScript.

---

#### Hướng dẫn chạy dự án Frontend (chocu-fe)

1. Cài đặt dependencies:
   - Mở terminal trong dự án và chạy lệnh:
     ```
     npm install
     ```

2. Chạy ứng dụng:
   - Sau khi cài đặt các gói, chạy ứng dụng bằng lệnh:
     ```
     npm run dev
     ```

3. Kiểm tra giao diện:
   - Sau khi ứng dụng chạy, bạn có thể truy cập vào địa chỉ `http://localhost:3000` để kiểm tra giao diện và các tính năng của ứng dụng.
